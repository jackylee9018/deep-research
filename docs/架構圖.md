企業用戶透過**單一工作台入口**使用 AI 能力；對話與日程分別由專用系統承載，研究與簡報在本專案內完成。

**埠與服務對照**

| 服務 | 預設埠 | 協議 | 說明 |
|------|--------|------|------|

| vLLM | 8000 | HTTP `/v1` | `OPENAI_ENDPOINT` |

## 6. 核心資料流

### 6.1 對話流（→ Open WebUI）

```mermaid
sequenceDiagram
    actor U as 用戶
    participant H as 首頁 /
    participant C as /api/app-config
    participant O as Open WebUI

    U->>H: 輸入提示詞 Enter
    H->>C: GET app-config
    C-->>H: openWebUIUrl
    H->>O: 302/replace ?q=提示詞&model=...
    O->>O: 建立對話 · 呼叫已配置模型
    Note over O: 模型可為 vLLM 本地端點
```

### 6.2 Deep Research 流

```mermaid
sequenceDiagram
    actor U as 用戶
    participant R as /research UI
    participant API as POST /api/research
    participant DR as deep-research.ts
    participant LLM as vLLM
    participant TV as Tavily

    U->>R: 主題 + breadth/depth
    R->>API: 串流請求
    loop 每層 depth
        DR->>LLM: 產生 SERP 查詢
        DR->>TV: 並行搜尋
        TV-->>DR: 結果
        DR->>LLM: 萃取 learnings
    end
    DR->>LLM: 生成 Markdown 報告
    API-->>R: SSE progress + report
    R->>U: 顯示 · 匯出
```

### 6.3 PPT Agent 流

```mermaid
sequenceDiagram
    actor U as 用戶
    participant P as /ppt UI
    participant OL as /api/ppt/outline
    participant GN as /api/ppt/generate
    participant GR as LangGraph
    participant EX as /api/ppt/export
    participant LLM as vLLM

    U->>P: 主題 · 附件
    P->>OL: 串流大綱
    OL->>LLM: 大綱生成
    OL-->>P: outline deck
    U->>P: 確認大綱
    P->>GN: 啟動 job
    GN->>GR: contentPlanner
    GR->>LLM: deck plan
    GR->>GR: validateDeckPlan
    GN-->>P: jobId · preview URL
    U->>P: /ppt/preview 編輯
    U->>EX: 匯出 PPTX
    EX-->>U: deck.pptx 下載
```

## 10. 能力邊界矩陣

| 能力 | 實作位置 | 本專案是否實作引擎 | 典型依賴 |
|------|----------|-------------------|----------|
| 日常對話 | Open WebUI | 否（僅跳轉） | `OPENWEBUI_URL` |
| 語音 / Call | Open WebUI | 否 | `?call=true` |
| 深度研究 | 本專案 `/research` | 是 | vLLM + Tavily |
| 簡報生成 | 本專案 `/ppt` | 是 | vLLM + PptxGenJS |
| 日程管理 | 雲端 SaaS | 否（入口規劃中） | SaaS URL |
| 模型推理 | vLLM（主） | 透過 OpenAI SDK 適配 | `OPENAI_ENDPOINT` |
| 報告 PDF | 本專案 | 是 | Puppeteer + Chromium |

## 11. 圖例說明

| 線型 | 含義 |
|------|------|
| 實線 `-->` | 已實作、主要資料路徑 |
| 虛線 `-.->` | 可選、規劃中、或依環境配置 |
| 子圖 `subgraph Local` | 企業內網可托管 |
| 子圖 `subgraph Cloud` | 需出站或雲端 SaaS |
